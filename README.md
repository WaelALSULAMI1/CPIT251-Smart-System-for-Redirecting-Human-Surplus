# Smart System for Redirecting Human Surplus

A Java-based volunteer management system developed for the CPIT 251 Software Engineering course.

The system connects volunteers with suitable volunteer opportunities based on skills, availability, and location. It aims to reduce wasted human effort by helping volunteers find suitable tasks and helping organizations find temporary volunteers more easily.

## Project Description

Many people have free time and useful skills, but they may not know where to use them effectively. At the same time, volunteer organizations may need temporary volunteers but face difficulty finding suitable people.

This project provides a smart volunteer matching system that recommends suitable tasks for volunteers and supports basic volunteer opportunity management.

## Main Features

- Create volunteer profiles
- Update volunteer skills and availability
- Browse volunteer tasks
- View recommended tasks
- Apply for tasks
- Post volunteer opportunities
- Match volunteers with suitable tasks
- Store volunteer, task, match, and application records
- Generate output files
